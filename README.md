# Supplier Product Harmonisation System

An automated data engineering pipeline designed to ingest, harmonise, and standardise supplier product catalogs from multiple European sources into a unified structure.

##  Project Structure
* **backend/**: FastAPI application, data extraction scripts, and database models.
* **frontend/**: Streamlit-based customer catalog dashboard.
* **data/**: Storage for raw and processed supplier data.
* **docs/**: Project architecture, API documentations, and notes.

##  How to Run Backend Local Setup
1. Navigate to backend: `cd backend`
2. Activate Virtual Environment: `.\.venv\Scripts\activate`
3. Run Starter Script: `python main.py`
